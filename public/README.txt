Place the custom app logo image here as `logo.png`.

- Recommended size: 256x256 (square) or at least 64x64.
- File name: logo.png
- Path after copy: web/public/logo.png

Vite will serve files in `public/` at the site root, so the app references this image at `/logo.png`.

If you prefer bundling the image, place it under `web/src/assets/` and update `App.jsx` to import it instead.
